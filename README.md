# Портфолио 
## Игры (LimeCore Games)
  1. Штурмовые псы: защита в пустыне.
     Штурмовые псы: защита в пустыне — напряжённая защита башни в суровых песках Готовы проверить свои стратегические навыки в борьбе с ордами агрессивных псов? В игре «Штурмовые псы: защита в пустыне» вам предстоит защитить свою базу в безжалостной пустыне от семи видов противников — каждый со своими уникальными способностями и тактикой. Ссылка на скачивание: https://www.rustore.ru/catalog/app/com.limecoregames.assaultdogs
  2. Морковка в лабиринте загадок
     Морковка в лабиринте загадок - это увлекательный 2D платформер-головоломка, который бросает вызов игрокам, предлагая им преодолеть хитроумный лабиринт, полный опасностей и загадок. Ссылка на скачивание: https://www.rustore.ru/catalog/app/com.limecoregames.carrotinthelabyrinth
  3. Жми на цвет!
     Простая, но захватывающая аркада для тренировки реакции и внимания. На экране появляются круги разных цветов — тебе нужно кликать только по кругу того цвета, который указан вверху. Ссылка на скачивание: https://www.rustore.ru/catalog/app/com.limecoregames.noname
